# Анализатор угроз Suricata (Suricata Threat Intelligence Analyzer)

Инструмент на Python для анализа логов **Suricata**, извлечения артефактов сетевой активности и обогащения их данными из внешних сервисов **Threat Intelligence** — **VirusTotal** и **Vulners**.

Скрипт автоматически извлекает индикаторы компрометации (IoC) из логов Suricata, проверяет их через API сервисов безопасности и формирует отчёты для последующего анализа и визуализации.

---

# Возможности

* анализ логов **Suricata (`eve.json`)**
* автоматическое извлечение артефактов:

  * IP-адресов
  * доменов
  * URL
  * сигнатур IDS
* фильтрация нерелевантных сигнатур Suricata
* игнорирование **LAN-адресов (RFC1918)**
* проверка индикаторов через:

  * **VirusTotal API**
  * **Vulners API**
* формирование отчётов:

  * полный отчёт
  * краткий статистический отчёт
* возможность построения графиков по результатам анализа

---

---

# Как работает программа

1. Suricata записывает события в файл `eve.json`
2. Скрипт анализирует файл и извлекает артефакты
3. Для найденных индикаторов выполняются запросы к:

   * VirusTotal
   * Vulners
4. Результаты сохраняются в JSON-отчёты
5. Данные могут использоваться для построения графиков и аналитики

---

# Извлекаемые артефакты

Скрипт автоматически собирает:

* IP-адреса источника
* IP-адреса назначения
* DNS-домены
* HTTP-домены
* URL
* сигнатуры IDS

При этом:

* **частные IP-адреса (LAN)** не отправляются на проверку
* нерелевантные сигнатуры Suricata отфильтровываются

---

# Используемые сервисы Threat Intelligence

## VirusTotal

Используется для анализа:

* IP-адресов
* доменов
* URL

Сохраняется статистика детектов:

* malicious
* suspicious
* harmless
* undetected

---

## Vulners

Используется для анализа **сигнатур Suricata**.

Позволяет находить:

* CVE
* security advisories
* описания уязвимостей
* информацию об эксплойтах

---

# Генерируемые отчёты

## artifacts_result.json

Содержит найденные артефакты.

Пример:

```json
{
  "all_ips": ["8.8.8.8"],
  "domains": ["example.com"],
  "urls": ["example.com/index.html"],
  "signatures": ["ET MALWARE Suspicious IP Connection"]
}
```

---

## full_report.json

Полный отчёт содержит:

* проверенные артефакты
* статистику VirusTotal
* полный ответ API
* результаты поиска Vulners

Используется для **глубокого анализа инцидентов**.

---

## stats_report.json

Краткий статистический отчёт.

Содержит только статистику детектов VirusTotal.

Пример:

```json
{
  "ips": [
    {
      "artifact": "8.8.8.8",
      "malicious": 3,
      "suspicious": 1,
      "harmless": 70,
      "undetected": 2
    }
  ]
}
```

Используется для:

* построения графиков
* аналитики
* ранжирования артефактов

---

# Визуализация результатов

На основе `stats_report.json` можно строить графики.

Например:

* количество детектов VirusTotal по IP
* сравнение verdict для доменов
* ранжирование наиболее подозрительных артефактов

Используемые библиотеки:

* pandas
* matplotlib
* seaborn

# Настройка API

Перед запуском необходимо задать ключи API.

Linux / macOS:

```
export VT_API_KEY="your_virustotal_key"
export Vuln_API_KEY="your_vulners_key"
```

Windows:

```
set VT_API_KEY=your_virustotal_key
set Vuln_API_KEY=your_vulners_key
```

# Ограничения

* бесплатный API VirusTotal имеет ограничения на количество запросов
* результаты Vulners зависят от совпадения сигнатуры
* анализ больших логов может занимать значительное время
